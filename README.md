# react-demo 开发模板

## 安装
1. git clone 代码到本地
2. npm i 安装
3. npm run start 运行
4. npm run mock 运行mockjs本地模拟服务器

分别执行以下命令
1. `git clone https://github.com/chkui/uepay-website-demo.git`
1. `$ cd  uepay-website-demo/client`
1. `$ npm i # 必须有nodejs环境`
1. `$ npm run dev`
